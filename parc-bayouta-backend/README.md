# Parc Bayouta Backend

This is the Node.js backend for the Parc Bayouta project.

## Structure

- `src/app.js`: Application setup (Express, middleware).
- `src/server.js`: Entry point.
- `src/config`: Configuration files.
- `src/controllers`: Request handlers.
- `src/routes`: API route definitions.
- `src/services`: Business logic.
- `src/middleware`: Custom middleware (error handling, etc.).
- `src/utils`: Utility classes and functions.

## Commands

- `npm run dev`: Run in development mode with nodemon.
- `npm start`: Run in production mode.

## API Endpoints

- `GET /api/health`: Health check.
